Standby

Home standby — Max FDP is the more limiting of report-time and SBY-start bands, unless the night SBY rule applies. FDP starts at report time if SBY duration is under 6h, or 6h after SBY start if 6h or more. Airport standby — FDP runs from the start of the standby.

Inputs

hm
Tick "2 hours notice" only if the callout happened between 2200 and 0800 local time — this applies the night SBY rule, where Max FDP is based on report time only. Leave it unticked for all other callouts: Max FDP is then the more limiting of the report-time and SBY-start bands. When not acclimatised, Table B (preceding rest) is used regardless of SBY start or report time — only the FDP start point depends on the 6-hour rule.

Delays7.3.5

For delays notified before leaving the place of rest. Delays of 10 hours or more, with no further disturbance, are treated as a rest period — use the Standard FDP calculator with the newly agreed report time in that case. Table A lookups use local time, converted from UTC using the "Local UTC offset" at the top of the page. All times are entered as UTC clock times with no calendar date — LOC results show a "+1 day" style badge when the result rolls past midnight.
